# iSensor Web Demo

iSensorApp과 **같은 화면 구성·에셋·카피**로 맞춘 mock 웹 시연본입니다. (BLE/서버 없음)

## 실행

```bash
cd web-demo
npm install
npm run dev -- --host
```

브라우저에서 로컬 URL을 열거나, 같은 Wi‑Fi의 Network URL을 QR로 공유하세요.

## 앱과 맞춰 둔 것

- Splash / Onboarding (main-logo, main-tire, CTA 문구)
- Home Axle (front/rear car, 타이어 카드, FL·FR·RLO… 라벨, Axle/Detail 탭)
- Tire Log (월 네비, hazard 배너, Issues 칩, 세션 카드)
- Insights (파란 헤더, damage, tire life, watch)
- Settings (Preferences / Support / Device Token 행)
- 하단 탭바 (앱 아이콘 + AI 버튼)
